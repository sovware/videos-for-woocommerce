== Frequently Asked Questions ==

= How does it work? =

It searches youtube using your product title and displays the searched result videos as related product videos in a nicely desiged grid.

= How much accuracy does it have? =

It completely depends on your product title. If your product title is too generic then you'll get generic videos.


== Installation ==

= Minimum Requirements =

* PHP 7.0 or greater is recommended
* WooCommerce 3.0 or greater is better
* WordPress 5.5 or greater is better

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handle the file transfer, and you won’t need to leave your web browser. To do an automatic install of Youtube Videos For WooCommerce, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
 
In the search field type “Youtube Videos For WooCommerce,” then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the Youtube Videos For WooCommerce plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
